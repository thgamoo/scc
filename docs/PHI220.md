# PHI 220 - Introduction to Symbolic Logic

- **Credits:** 3
- **SBC:** QPS
- **DEC:** C & Skill 1

## Description
Systematic techniques for assessing the validity of arguments: truth tables and truth values analysis, Venn diagrams, elementary quantification theory, and deduction in both the propositional calculus and quantification theory.

## Prerequisites
- Satisfaction of entry skill in mathematics requirement OR level 2+ on the mathematics placement examination
- **Advisory:** U2 standing or one course in philosophy

## Grade Requirement
A through C
